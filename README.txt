This software has only been tested with PostgreSQL and PHP 5.3.

To Use:

You will need to set up a database, and provide information in application/config/database.php:
- username
- password
- database name
- type of DBMS (mysql, mysqli, mssql, postgres, oracle, sqlite, odbc)

As of this 2011-11-20, CodeIgniter supports MySQL (4.1+), MySQLi, MS SQL, Postgres, Oracle, SQLite, and ODBC.

You will also need to use an SMS service. This site was originally developed to use Twilio.
